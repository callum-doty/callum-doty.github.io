Silver Rule

Limitations of the Golden Rule: 'Treat others the way you want to be treated.'

Published on June 15, 2025

Growing up, we were inundated with rules and maxims meant to guide our behavior and instill a sense of empathy. One such rule, echoed throughout our elementary school, was the Golden Rule: "Treat others the way you want to be treated." It was a mantra drilled into us, intended to foster kindness and consideration towards others.

But if I'm being honest, this rule always left me a bit perplexed. Deep down, I couldn't shake the feeling that not everyone wished to be treated in the same manner they treated themselves. There seemed to be a disconnect between the Golden Rule and human nature.

Fifteen years later, I can finally respond to this argument. Let's define the Golden Rule.

The golden rule is defined as: an act is right if and only if, in performing it, we treat others as we would like to be treated. The notions of necessary and sufficient conditions are defined:

  1. If we are acting rightly, then we're treating others as we would like to be treated.
  2. If we're treating others as we would like to be treated, then we're acting rightly.

Hmmm, it appears a loop is forming. Condition (1) supports the conclusion, Condition (2). Condition (2) assumes that we are acting rightly if we are treating others the way we would like to be treated. Wait a second — the conclusion just restated Condition (1)!

The conditions of the golden rule assume each other rather than using sound reasoning and evidence. It's called circular reasoning — a logical fallacy. Surprise! You have been guided by flawed logic your entire childhood.

The Conditions attempt to define moral rightness with fairness. If you think it is right to treat someone in a specific fashion, then you should accept that same treatment from the receiver. It really makes you think that the masochists in elementary school had the upper hand.

Suppose we create our own version of the Golden Rule. We could begin with an equation:

x = y

To be as concise and inclusive as possible, let's focus on individual factors, contextual factors, impact on others, and values. Assigning notation:

  • Individual factors = Σni=1 βi ⋅ fi
  • Contextual factors = γ ⋅ C
  • Impact on others = δ ⋅ D
  • Values = ε ⋅ V

The new formula:

X = Σni=1 βi ⋅ fi + γ ⋅ C + δ ⋅ D + ε ⋅ V

When we find X for a given action, we can determine whether the action is ethical (higher) or unethical (lower). This raises a few open questions:

  • How do we determine the weights (β, γ, δ, ε)? These vary significantly based on cultural and personal values.
  • We need boundary conditions or normalization to make the scale more interpretable.
  • Should there be a temporal component to account for both short-term and long-term consequences?
